SDLC PHASES

Concept An idea or thought, conceived in the abstract mind.
Inception The beginning of a great adventure, one's starting point when creating a software.
Iteration The process of repetition in the code in order to generate a sequence of outputs.
Release To let go, to be free. Software now being available for the public eye.
Maintenance Keeping the software in top shape (bug fixes, updates both major and minor, etc.)
Retirement Eventually, the software can no longer be supported in updates and must be put down like a sick dog.
