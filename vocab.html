Layer Layer is a set of hierarchical structure that gives separate functions or responsibilities within a system.
Application A program designed to handle specific tasks for user.
Software Software is a set of instructions, data, or program.
Logic Layer The part of the program classes determine the info needed to solve problems, manipulate the info, and return results to the presentation layer for formatting.
Authorization Allowed access for someone to get data or information for themselves.
User Interface The interaction between the human and communication.
Persistence Layer A way to save and retrieve data that the applications use.
Logging The process of recording or storing information and/or messages.
Error Handling The process of managing and addressing errors that have the possibility of occurring during the execution of the program.
Authentication an identification method, such as a password or unique identification code that is assigned to a user to identify themselves.
Cross-cutting concerns A concern or responsibility that affects multiple layers of code.
